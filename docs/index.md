## What these docs cover

This documentation explains how the Helix Genome Protocol works — from the mechanics of wrapping and unwrapping tokens, to deploying your own Genome as a creator. Whether you are a user looking to earn yield, a developer integrating with the protocol, or a creator building a new Genome, this guide covers everything you need. The docs are organized into Concepts (how it works), User Guides (how to use the app), Creator Guide (how to deploy and manage a Genome), and Reference (contract methods and FAQ).
